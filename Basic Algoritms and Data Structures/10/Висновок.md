# Висновок

Алгоритм перебору вкрай цінний для тих хто починає відкривати бізнес з невеликою еількістю ресурсів, тож завдяки цьому алгоритму можна аналізувати як збільшити парибуток мінімально витративши ресурси.

Метод Монте-Карло це про решення завдяки теорії ймовірності, коли нам паотрібно максимально швидко отримати результат нехтуючи точністю. Цей метод використовується для надто складних та нестандартних задач.ю
