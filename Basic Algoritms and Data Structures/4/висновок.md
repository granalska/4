# Висновок

Під час роботи було перевірено алгоритми сортування злиттям, вставкою та Timsort на масивах різного розміру.

Найшвидшим у всіх тестах виявився Timsort, оскільки він є вбудованим та добре оптимізованим алгоритмом.

Сортування злиттям працювало швидше за сортування вставкою, особливо на великих масивах.

Найповільнішим було сортування вставкою, яке погано підходить для великих обсягів даних.
