# Висновок

Під час виконання домашньої роботи я протестувала три алгоритми пошуку і з’ясувала, що абсолютним лідером є алгоритм Боєра-Мура. Він виявився найшвидшим на обох текстах, тому що вміє хитро «перестрибувати» через непотрібні літери, значно економлячи час.

Друге місце зайняв Кнут-Морріс-Пратт, який працює стабільно, але перевіряє символи підряд, тому трохи відстає.

Найповільнішим виявився Рабін-Карп, оскільки він витрачав зайвий час на математичні обчислення замість простого порівняння.

Алгоритми перевірялися на двох статтях:
- [Стаття 1](https://drive.google.com/file/d/18_R5vEQ3eDuy2VdV3K5Lu-R-B-adxXZh/view?usp=sharing)
- [Стаття 2](https://drive.google.com/file/d/18BfXyQcmuinEI_8KDSnQm4bLx6yIFS_w/view?usp=sharing)

Отже, мій експеримент показав, що для звичайного пошуку слів у тексті найефективніше використовувати саме метод Боєра-Мура.
